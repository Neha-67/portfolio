# Portfolio

This repository contains a working portfolio project:

- `projects/execution_coach/` — Execution Coach app built with Python and Streamlit

## Run the project

```bash
cd projects/execution_coach
pip install streamlit
streamlit run app.py
```

## Project contents

- `agents.py` — goal intake, planning, and evaluation logic
- `app.py` — Streamlit UI for morning/evening workflows and session history
- `main.py` — CLI version of the execution coach
- `requirements.txt` — required Python packages
- `README.md` — project documentation for the Execution Coach

## Notes for hiring managers

The Execution Coach project demonstrates:
- clear Python code structure
- a user-facing Streamlit interface
- session persistence and workflow state
- a clean repo layout with only relevant files
